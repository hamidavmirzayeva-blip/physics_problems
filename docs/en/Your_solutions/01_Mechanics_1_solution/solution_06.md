

We are given the **velocity function**:

v(t) = t² + 2t − 5

We are also told:

x(0) = 4 (the position at time 0).

We must find:

1️⃣ Position at **t = 3**
2️⃣ Acceleration at **t = 3**

---

# 1. Find acceleration

Important rule in physics:

acceleration = derivative of velocity

So we differentiate:

v(t) = t² + 2t − 5

Derivative rules:

* derivative of t² = 2t
* derivative of 2t = 2
* derivative of constant = 0

So:

a(t) = 2t + 2

Now substitute **t = 3**:

a(3) = 2(3) + 2
a(3) = 6 + 2
a(3) = **8**

✅ Acceleration at t = 3 is **8**.

---

# 2. Find the position function

Velocity is the derivative of position.

So to find position we **integrate velocity**.

v(t) = t² + 2t − 5

Integrate each term:

∫t² dt = t³ / 3

∫2t dt = t²

∫−5 dt = −5t

So position becomes:

x(t) = t³/3 + t² − 5t + C

C is a constant.

---

# 3. Find the constant C

We know:

x(0) = 4

Substitute t = 0:

4 = 0 + 0 − 0 + C

So:

C = **4**

Now the position function is:

x(t) = t³/3 + t² − 5t + 4

---

# 4. Find position at t = 3

Substitute t = 3:

x(3) = 3³/3 + 3² − 5(3) + 4

Calculate step by step:

3³ = 27

27 / 3 = 9

3² = 9

Now:

x(3) = 9 + 9 − 15 + 4

x(3) = 18 − 15 + 4

x(3) = 3 + 4

x(3) = **7**

---

# Final answers

Position at t = 3:

x(3) = **7**

Acceleration at t = 3:

a(3) = **8**
