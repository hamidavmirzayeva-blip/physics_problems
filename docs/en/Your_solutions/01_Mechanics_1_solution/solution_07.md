

We are given the parametric equations:

x(t) = 2t²
y(t) = 3t³

---

# 1. Eliminate the parameter t

First solve **x(t)** for (t).

x = 2t²

Divide by 2:

x / 2 = t²

Now take the square root:

t = √(x/2)

---

Now substitute this into **y(t)**.

y = 3t³

Replace (t):

y = 3(√(x/2))³

Now simplify:

(√(x/2))³ = (x/2)^(3/2)

So the trajectory equation is:

y = 3(x/2)^(3/2)

This is the **path of the object**.

---

# 2. Draw the trajectory (what the graph looks like)

The equation

y = 3(x/2)^(3/2)

means:

* when x = 0 → y = 0
* as x increases, y increases faster

So the curve:

• starts at the origin (0,0)
• curves upward to the right
• becomes steeper as x increases

---

# 3. Velocity vector

Velocity = derivative of position.

Position vector:

r(t) = (x(t), y(t))

Differentiate each component.

x(t) = 2t²
dx/dt = 4t

y(t) = 3t³
dy/dt = 9t²

So velocity vector:

v(t) = (4t , 9t²)

---

# 4. Speed |v(t)|

Speed is the magnitude of velocity:

|v(t)| = √[(4t)² + (9t²)²]

Square each term:

(4t)² = 16t²

(9t²)² = 81t⁴

So:

|v(t)| = √(16t² + 81t⁴)

You can also write:

|v(t)| = √(t²(16 + 81t²))

= |t| √(16 + 81t²)

---

# 5. Acceleration vector

Acceleration = derivative of velocity.

v(t) = (4t , 9t²)

Differentiate:

d/dt(4t) = 4
d/dt(9t²) = 18t

So:

a(t) = (4 , 18t)

---

# 6. Magnitude of acceleration

|a(t)| = √(4² + (18t)²)

Calculate squares:

4² = 16

(18t)² = 324t²

So:

|a(t)| = √(16 + 324t²)

---

# 7. Is acceleration constant?

Acceleration vector:

a(t) = (4 , 18t)

The second component **18t changes with time**.

So the acceleration **depends on t**.

Therefore:

❌ Acceleration is **not constant**.

---

# Final answers

Trajectory:

y = 3(x/2)^(3/2)

Velocity:

v(t) = (4t , 9t²)

Speed:

|v(t)| = √(16t² + 81t⁴)

Acceleration:

a(t) = (4 , 18t)

Acceleration magnitude:

|a(t)| = √(16 + 324t²)

Acceleration constant?

❌ No, because it depends on **t**.
