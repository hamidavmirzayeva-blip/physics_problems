

---

# 1. Start with the range formula

The horizontal range R of a projectile is:

R(θ) = (v0² * sin(2θ)) / g

* v0 = initial speed (constant)
* g = gravity (constant)
* θ = launch angle (variable)

---

# 2. Idea

To find the **maximum**, we use calculus:

1. Take the derivative of R(θ) with respect to θ
2. Set it equal to 0
3. Solve for θ

---

# 3. Differentiate

R(θ) = (v0² / g) * sin(2θ)

* v0² / g is constant
* Derivative of sin(2θ) with respect to θ is 2 * cos(2θ)

So:

dR/dθ = (2 * v0² / g) * cos(2θ)

---

# 4. Solve for maximum

Set derivative = 0:

(2 * v0² / g) * cos(2θ) = 0

* The constant (2 * v0² / g) ≠ 0
* So we must have cos(2θ) = 0

---

# 5. Solve cos(2θ) = 0

Cosine is zero at 90°, 270°, …
Take the first solution:

2θ = 90°
θ = 45°

---

# ✅ Conclusion

The **maximum range** occurs when the **launch angle θ = 45°**.

---

💡 **Tip / Intuition**

* If angle < 45° → too flat → doesn’t go high enough → shorter distance
* If angle > 45° → too steep → goes high but not far → shorter distance

45° gives the **perfect balance** between height and distance.
