# 1. What the position vector means

The position of the object is

r(t) = (3t²) i + (5t − 8t²) j

This tells us the **coordinates of the object at time t**.

So we can read it as:

x(t) = 3t²
y(t) = 5t − 8t²

Meaning:

Position = ( x(t), y(t) )

---

# 2. What is velocity?

Velocity tells us **how fast the position changes**.

In calculus:

velocity = derivative of position

v(t) = r'(t)

So we differentiate **each component separately**.

---

# 3. Differentiate the x-component

x(t) = 3t²

Derivative rule:

d/dt (t²) = 2t

So

d/dt (3t²) = 6t

So the x-velocity is:

vx(t) = 6t

---

# 4. Differentiate the y-component

y(t) = 5t − 8t²

Derivative:

d/dt (5t) = 5
d/dt (−8t²) = −16t

So

vy(t) = 5 − 16t

---

# 5. Velocity vector

Now combine both components:

v(t) = (6t)i + (5 − 16t)j

This is the **velocity vector**.

---

# 6. What is acceleration?

Acceleration is **how velocity changes**.

So we differentiate velocity.

a(t) = v'(t)

---

# 7. Differentiate velocity

Velocity:

v(t) = (6t)i + (5 − 16t)j

Differentiate each part.

First component:

d/dt (6t) = 6

Second component:

d/dt (5 − 16t)

Derivative of 5 = 0
Derivative of −16t = −16

So

= −16

---

# 8. Acceleration vector

a(t) = 6i − 16j

---

# Final Answers

Velocity vector:

v(t) = (6t)i + (5 − 16t)j

Acceleration vector:

a(t) = 6i − 16j
