 We’ll find all local maxima and minima for:

**f(x) = 3x² − 12x + 7**

---

### **Step 1: Find the derivative**

Why?

* The derivative, f'(x), gives the **slope** of the function at any point.
* At a **local maximum or minimum**, the slope is **0** (the graph is flat).

**Rules to take derivatives:**

1. The derivative of xⁿ is n * xⁿ⁻¹
2. The derivative of a constant (like 7) is 0
3. The derivative of k*x is k

Now apply it to each term:

* 3x² → derivative is 2 * 3 * x = 6x
* −12x → derivative is −12
* 7 → derivative is 0

So:

**f'(x) = 6x − 12** ✅

---

### **Step 2: Find critical points**

Critical points are where **f'(x) = 0**.

Set the derivative equal to 0:

6x − 12 = 0

Add 12 to both sides:

6x = 12

Divide both sides by 6:

x = 2

✅ So the critical point is **x = 2**

---

### **Step 3: Determine if it’s a maximum or minimum**

We use the **second derivative test**:

* Find the second derivative f''(x)
* If f''(x) > 0 → minimum
* If f''(x) < 0 → maximum

Take the derivative of f'(x) = 6x − 12:

f''(x) = 6

* 6 > 0, so the graph **opens upward** → the critical point is a **local minimum**

---

### **Step 4: Find the y-value**

We know x = 2. Find f(2):

f(2) = 3*(2)² − 12*(2) + 7

f(2) = 3*4 − 24 + 7

f(2) = 12 − 24 + 7 = −5

✅ So the point is (2, −5)

---

### **Step 5: Conclusion**

* **Local minimum:** (2, −5)
* **Local maximum:** none (because the parabola opens upward)
